About me

I'm an Aerospace Engineer with a strong passion for all all kind of technologies.
Born in Montreal, raised in southern Italy and lived in France for a while, I have studied and worked as an engineer across three countries. I started my career writing code for flight simulators, which somehow helped me finding my way into flight testing. After three years of crunching code on "grounded cockpits”, I decided to go back to school and pursue a one year Master in flight testing, a great experience that has got me working for somewhat more than three years in France. While not at work I love getting my hands dirty on self-started projects, going from web technologies, hardware tinkering down to wood crafting. Being a famous rock guitarist, a fighter pilot or simply one of the A-Team's has always been my childhood dream. Beside my professional accomplishments the things I really couldn't live without are my sweet daughter and my stellar wife.
I am currently living in Toulouse, France.

Projects

Academic

Validation of a Pitot Heat Computer (PHC)

Internship Project with Airbus | Assystem, September 2012

A complete system validation cycle has been carried out by testing and analyzing a new software standard of PHC (Probe Heat Computer) for Airbus Single Aisle products. The activities included lab testing, flight data analysis and redaction of a test report. Results are then used by Airbus in order to demonstrate compliance with design requirements and certification specification.

Design of an Autopilot for a Flight Simulator

Master's Final Project | Concordia University, July 2009

The idea of this project is to use a functional Beechcraft 200 flight simulator as a platform to implement and test autopilot control laws. The design of the control laws is performed in Simulink while the implementation on the simulator is done in C++. Classical linear control techniques are used to design control laws.

Specification and Selection of a Brake Control System

Case Study with Bombardier | McGill University, December 2008

The goal of this project is to define brake control system specification, present certification compliance plan and perform life cycle cost analysis, while simultaneously considering safety, performance and cost. Tools and techniques such as MMEL, FTA, and LCC analysis were used for the realization of the product.

Computer vision controlled rover-robot

Mechtronics & Automation Final Project | Concordia University, May 2008

This project consisted in designing and developing a basketball playing rover-robot which is able to perform two tasks: "dunking" and throwing a ping-pong ball in a net by means of autonomous control. The main control feature is a vision system made of a top viewing camera that enables computer vision algorithms written in C to process and control the robot actions.

GPS simulation with MATLAB

Avionics Navigation Systems Final Project | Concordia University, April 2008

The main task of this project was to compute the XYZ position of a user by processing navigation messages from six GPS satellites. All the data and all the information necessary to develop the algorithm were provided during the course. The main goal of this project was to demonstrate GPS principles by using simulation tools such as Matlab.

F.E.M simulation with MATLAB

Final Element Methods Final Project | Concordia University , August 2007

A Matlab program that conducts the assembly of element matrices, imposition of boundary conditions and partitioning. A finite element methods algorithm finds the system solution and post-processing calculates the axial stresses and strains, principal stresses and strains, and Von Mises stresses in the element at any cross-section.

F.E.M. analysis of composite panels under buckling

Bachelor's Final Project | University of Palermo , July 2006

A parametric study that aims at understanding how buckling of laminated composite panels behave under simple axial compression loads. Different fibers orientation, sequence of lamination, sizes and boundary conditions are examined by means of F.E.M. simulations ran in Patran.
